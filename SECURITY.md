# Security

请不要在公开 issue 中提交令牌、凭据、私人桌宠文件或其他敏感信息。

Skill Key 会保存在当前用户的 `~/.codex/pet-club/config.json` 中。不要把这个文件提交到 Git、发送给他人或粘贴到 issue。CLI 不会在正常输出中显示完整 Key；如怀疑泄露，请立即在桌宠库账户页面撤销该 Key。

安装与投稿命令会拒绝绝对路径、路径穿越、重复 ZIP 路径、超限文件、校验和不匹配和不合规的 Codex v2 桌宠包。发现安全问题时，请通过 GitHub 的私密漏洞报告功能联系维护者。

Skill 自动更新只接受本项目官方 GitHub Release 的固定 URL 格式，并同时校验版本号、归档大小和 SHA-256。更新包会拒绝路径穿越、重复路径、符号链接、超限展开和缺失运行文件；源代码工作区不会被自动替换。更新采用同一磁盘上的事务切换，旧目录只在切换过程中临时存在，成功后立即删除，失败时恢复当前可用版本。
